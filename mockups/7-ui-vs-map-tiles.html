great — earthy greens, more illustrated feel
https://tiles.stadiamaps.com/tiles/stamen_terrain/{z}/{x}/{y}{r}.png
Hand-drawn-feeling earthy greens + cream. Free Stadia API key required.

OSM, CartoDB Positron, and OpenTopoMap: no API key required, free. Stadia Stamen Terrain: free tier available at stadiamaps.com — requires API key appended as ?api_key=YOUR_KEY. OpenTopoMap tiles can be slower at high zoom; best at city/neighbourhood scale.
